Roadmap

The ResearchSpace project has the following 8 month roadmap from December 2016:


Current Functionality

  • Clipboard v1 - save items to the clipboard, use them to filter searches, and use in arguments and visualisations
  • Semantic Narratives and Annotations v1 - create narratives with embedded semantic information
  • Arguments & Assertions v1 – use ResearchSpace entities to make assertions of belief and arguments
  • Semantic Web database v1 – Build dynamic Semantic Web databases
  • Image Annotation with IIIF v1 – Annotate high resolution images within the IIIF environment
  • Lightbox v1 – view and manipulate images

January 2017

  • Drag and Drop Clipboard v2 - a more accessible and graphical version with tagging facilities
  • Semantic Narratives and Annotations v2 - enhanced version with improved presentation and visualisations
  • Enhanced Security

June 2017

  • Arguments & Assertions v2 – enhanced functionality and integration with Semantic Web Database builder
  • More integrated datasets
  • General enhancements to all components

The date for the release of the project source code is TBC.
