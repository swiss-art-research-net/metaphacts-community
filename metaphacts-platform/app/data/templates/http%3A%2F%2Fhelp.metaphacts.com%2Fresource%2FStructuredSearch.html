Structured Search

The platform provides a powerful mechanism for expressing the userâ€™s request in an incremental way: structured search based on fundamental categories and relations (semantic-search-query-builder). Structured search assumes that the user wants to search for data instances classified into some fundamental categories: the most representative concepts within the domain.

Configuration

semantic-search-query-builder defines the default resource-selector configuration using Lookup Service technology. This abstraction allows database independent configuration of the search for the default case, i.e. the platform makes sure to select a lookup service implementation optimized for the target database.
