CSS Layout

The layout frameworks help manage the layout of pages. They allow dividing HTML-elements into grids of rows and columns.

Bootstrap Layout

To create a basic grid system one needs to use bs-row and bs-col components with xs, sm, md, lg attributes. See Bootstrap Grid to learn more about grid system.

Use Bootstrap Layout to create a responsive layout based on the 12-columns grid system.

Built-in CSS classes

The platform carries a set of CSS classes that help layout a page header and body. Use page__header, page__header-thumbnail and page__header-body to separate out your header content and page__body class to place tha page body content.

Workspace Layout

Helps creating complex workspace layouts by combining mp-workspace-layout, mp-workspace-column, mp-workspace-row, mp-workspace-item components. The workspace components also come with resizable spaces and an option to collapse side panels completely from view, which delivers more flexibility to the end-user if compaired to CSS layouting solutions.
