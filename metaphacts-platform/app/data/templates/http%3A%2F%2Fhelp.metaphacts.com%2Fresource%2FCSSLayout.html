CSS Layout

The layout frameworks help to manage the layout of pages. They allow dividing HTML-elements into grids of rows and columns.

Bootstrap Layout

To create a basic grid system one need to use bs-row and bs-col components with xs, sm, md, lg attributes. See Bootstrap Grid to learn more about grid system.

Use Bootstrap Layout to create a responsive layout based on the 12-columns grid system.

Flex Layout

To use Flex Layout one need to add data-flex-layout and data-flex-self attributes to HTML-elements. See Flex Layout Attribute for more details and configuration options.
