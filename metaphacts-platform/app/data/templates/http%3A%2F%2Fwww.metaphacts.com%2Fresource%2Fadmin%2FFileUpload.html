File Upload BETA

Please note:
  • Uploads may silently overwrite existing files. For the time being, there is no additional warning or check.
  • Uploads to any of these directories may change the system behaviour, i.e. if you upload / overwrite a config file, this may possibly even break the system.
  • It is recommended to limit the user group which may perform such low-level file operations, i.e. in the default configuration users with the "admin" role have access to the upload functionality.
  • It is currently not possible to list existing files or delete files from the respective directories, i.e. after a successful upload you should copy the relative upload path for your reference.
