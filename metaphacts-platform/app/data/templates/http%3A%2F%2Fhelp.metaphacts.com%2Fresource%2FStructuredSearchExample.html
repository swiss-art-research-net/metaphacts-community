Try to find a person by date of birth.

  • Select the search category Person
  • Select the related category Time
  • Select the relation type birth date
  • Select the range 1990 - 2000

Now, try to narrow down the result set using facets.

  • With the birth date facet, try to narrow down the date range.
  • With the height facet, select the interval of height values.
  • With the knows and interested in facets select the results which satisfy other special criteria.
