Customizing the page layout (e.g. Header)

The platform supports customizing various parts of the page layout, including header, footer and login page.

This can be achieved by providing a handlebars HTML template to the config/page-layout folder of the app, e.g. config/page-layout/header.hbs for the header. Typical customizations include the addition of a logo or modifications of the menu.

Remarks:

  • the contents of the HTML template must be in-line with the pre-bundled versions of the product
  • it is recommended to upgrade any customized template when upgrading the version of the platform
