[[!-- Ontodia Toolbar actions included on Help:Ontodia --]]
  • save - save diagram;
  • forceLayout - perform Force layout;
  • clearAll - delete all elements from diagram;
  • zoomIn - zoom in diagram;
  • zoomOut - zoom out diagram;
  • zoomToFit - zoom to fit diagram;
  • exportPng - export diagram as PNG;
  • exportSvg - export diagram as SVG;
  • print - print diagram.
