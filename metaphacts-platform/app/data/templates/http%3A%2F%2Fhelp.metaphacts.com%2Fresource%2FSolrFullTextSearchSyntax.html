Full-Text Search Syntax

The full-text search supports a special query syntax to express complex queries.

Searching for a single keyword

By default, the system looks for exact matches of provided keywords. For example, searching for an Actor by the keyword michel will return e.g., "Michel Molart" and "Michel Cazin", but not "Michelangelo".

Keywords with wildcards

The system supports single and supports multiple character wildcard searches within single terms.

For example, searching for mar? will return actors with names spelled as "Mark" and "Marc"

To perform a multiple character wildcard search use the "*" symbol. Multiple character wildcard searches looks for 0 or more characters.

For example, searching for michel* will return both "Michel Molart" and "Michelangelo".

Combining multiple keywords

For a query containing two keywords, the system will return results matching both the first AND the second one.

For example, searching for captain cook will include "Captain James Cook", but not "Elizabeth Cook" or "Captain John van Gelder".

In order to get the results matching either of the two keywords, one has to use the conjunction OR. Search for captain or cook returns multiple results including either "Captain" or "Cook"

Exact phrase match

Using quotes "" enclosing several keywords tells the system to search for an exact match of the whole phrase.

For example, searching for british museum will return both "The British Museum" and "British Salonika Force Museum". But searching for "british museum" will only retrieve the first of the two.

Excluding keywords

You can use the modifier NOT or the minus sign - to specify the keywords which should be excluded from the results.

For instance, a query cook will return "Elizabeth Cook" in the results, but cook -elizabeth will exclude it from the result set.

Grouping

You can use parentheses () to group clauses to form sub queries. This can be very useful if you want to control the boolean logic for a query.

For example, a query for things (xiv or xv) and louis will return results mentioning both Louis XIV and Louis XV.
